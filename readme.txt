# Brick Breaker Game

Brick Breaker is a classic arcade-style game developed using the Pygame library. The objective of the game is to control a paddle to bounce a ball and break bricks in the play area.

## Features

- Intuitive controls: Move the paddle left and right to keep the ball in play.
- Score tracking: Keep track of your score as you break bricks.
- Lives system: You have a limited number of lives to complete the game.
- Win and lose screens: See special screens when you win or lose the game.

1. Make sure you have Python and Pygame installed on your computer.

5. Use the left and right arrow keys to control the paddle and keep the ball in play. Break all the bricks to win each level.

## Game Controls

- Left Arrow: Move the paddle to the left.
- Right Arrow: Move the paddle to the right.
- Mouse Click: Navigate the main menu and retry/return to the main menu in case of a win or loss.
